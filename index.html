Introdução


O que são Matrizes?

"São tabelas que organizam valores numéricos relacionados a determinada situação."- Lessa, V. L. Sendo assim veremos a seguir alguns exemplos de operações matemáticas com matrizes feitas em Java Script.

O meu Trabalho.
I. Matrizes Fixas

As matrizes fixas são estabelecidas no próprio código, ou seja, não podem ser alteradas a não ser que se mude os valores pré estabelecidos no código. As operações feitas com essas matrizes foram as de adição e subtração.


II. Matrizes "Math.rand"

As matrizes do método "Math.rand" são sempre diferentes toda vez que a página é recarregada. As operações feitas são de adição e subtração.


III. Matrizes Geradas pelo Usuário

Essas matrizes são digitadas por você por meio de "avisos" com digitação. As operações também para adição e subtração.


IV. Matrizes geradas pela digitação de números pelo usuário de Multiplicação

Essas matrizes são digitadas por você por meio de "avisos" com digitação. As operações também para multiplicação.
